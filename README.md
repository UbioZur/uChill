# *uChill* - KDE Atomic Desktop

## Introduction

This repository serve as my deployment pipeline for my custom KDE Atomic Desktop *uChill*.
Base on uBlue and Fedora, it ams at providing a base OS with minimal installed applications.

Feel free to explore, learn and copy parts for your own. Enjoy!



## uBlue OS

I was only able to do this thanks to the work of [uBlue-OS](https://github.com/ublue-os), and using their [base-main](https://github.com/ublue-os/main/pkgs/container/base-main) image.

You can also check their [image-template](https://github.com/ublue-os/image-template) repository to create your atomic desktops.
